# Backend Mastery — Practice & Projects

This repo is a collection of backend and full-stack projects I built while deepening my skills across the Node.js ecosystem — from core fundamentals up through full project builds with authentication, payments, caching, and real-time systems.

## 🧰 Tech Covered

- **Runtime/Framework:** Node.js, Express.js
- **Databases:** MongoDB (Mongoose), MySQL
- **Auth:** JWT, session-based (stateful) and stateless auth, cryptography basics
- **Caching:** Redis
- **Real-time:** WebSockets
- **APIs:** REST, GraphQL
- **Payments/Media:** Stripe, Cloudinary
- **Frontend (for full-stack projects):** React
